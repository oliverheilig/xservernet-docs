Ptv.XServer.Demo.Geocoding Namespace

 
Classes
  ClassDescription
DataContainerT
Data container class for exchanging data between the GeocodeResultsLayer and the business logic code.
GeocoderBase
Base class for geocoding.
GeocoderDemo

Demonstrates the usage of the xLocate server. To support the AutoCompleteTextBox control, this class exposes some public properties. Use AddTo() and Remove() to add or remove a geocoder to/from the map. The Remove() call is essential for cleaning up resources.

See the Geocode an Address topic for an example.

GeocodeResultsLayer
Layer displaying the geocoding results.
GeocoderTemplateSelector
Selector of the data template which is used for the AutoCompleteTextBox. Either a wait message or the geocoding results can be displayed there.
MultiFieldControl

Control which offers a multi field geocoding. The geocoding request address is entered field by field.

See the Geocode an Address topic for an example.

MultiFieldData
Address data which can be entered for a multi field geocoding.
MultiFieldGeocoder
Demonstrates how the geocoding of a multi field input can be implemented.
SingleFieldControl

Control which offers a single field geocoding. The search address is entered as a single string containing all information.

See the Geocode an Address topic for an example.

SingleFieldGeocoder
Demonstrates how the geocoding of a single field input can be implemented.
