Overview

PTV xServer .NET is a SDK which adds xServer functionality to .NET client applications. The primary component is the WpfMap control, which allows building interactive map applications with PTV xServers. While the technology of this control is based on Windows Presentation Foundation, the SDK also provides a FormsMap control for easy Windows Forms integration.

The SDK comes with a set of sample use cases which demonstrate the use of the SDK for typical scenarios.

Refer to the API documentation for detailed information on classes, methods and parameters.

More information about the basic concepts of this SDK can be found in the Concepts chapter.

Prerequisites

PTV xServer .NET and/or its demos require

  • at least Microsoft Windows Vista. Microsoft Windows 7 or Microsoft Windows 8 is recommended.

  • at least Microsoft .NET 3.5 for directly running the use cases provided through the Demo Center.

  • at least Visual Studio C# 2008 Express Edition for developing applications based on PTV xServer .NET. With regard to the WPF integration and the overall development experience, Visual Studio 2010 or later is recommended.

  • internet access as PTV xServer .NET incorporates PTV xServer running on Azure.

All third-party libraries used by PTV xServer .NET and its Demo Center are included in the SDK package.

PTV xServer

PTV xServer .NET incorporates PTV xServer running on an Azure test environment. Because this system may be used for demo purposes only, own services/installations will be set up when developing applications based on PTV xServer / PTV xServer .NET.

Many different deployment scenarios are valid for PTV xServer when being used through PTV xServer .NET or Microsoft .NET in general. Please refer to the PTV xServer documentation or contact PTV in case there are any questions regarding PTV xServer deployment.

PTV xServer .NET was developed on the base of PTV xServer 1.18, but should be able to run with newer version of PTV xServer. PTV xServer .NET uses the following servers:

  • PTV xMap Server

  • PTV xLocate Server (Demo Center)

  • PTV xRoute Server (Demo Center)
