FeatureLayerUseCase Class

This use case encapsulates all relevant arrangements necessary for rendering the Feature Layer attributes and use them for influencing the route calculation. Additional data is necessary for a proper functioning, therefore this use case must be configured according the available Feature Layer data. Three different ways are shown how the configuration can be made, see FeatureLayerUseCase constructor.
