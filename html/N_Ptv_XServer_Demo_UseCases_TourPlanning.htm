BusinessToXB, K
A helper class which maps business objects (usually identified by a unique string) To xServer objects identified by an int. Note: The mapping is not unique! If you create it repeatedly, you get different IDs. Remind this if you need to work with persistent xTour objects, for example if you re-use distance matrices.
