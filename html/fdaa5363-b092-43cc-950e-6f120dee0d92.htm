Set the Map Location

This page shows how to set the map location. The map location defines the part of the map which is visible in the map window.

This page contains the following sections:

  • How to set the map location
How to set the map location

The demo center sample shows how to set the map location. When starting the demo application, some buttons on the upper left are shown, by which zoom in and out of the map and move the map up, down, right and left is possible. These are the navigation buttons.

Map with navigation buttons on the upper left and zoom slider on the right.

One option to set the map location to a new value is to use manually those navigation buttons or the zoom slider. But it is possible to do this by code. Setting a new map location is possible using the SetXYZ(Double, Double, Double, Boolean) method. Below, see the code for actions on the different buttons.

Moving the map 25% up:

C#
Copy
// move the current map location by 25 %
double dY = MapView.FinalEnvelope.Height * 0.25;

// Setting the new map location.
MapView.SetXYZ(MapView.FinalX, MapView.FinalY + dY, MapView.FinalZoom, Map.UseAnimation);

Moving the map 25% down:

C#
Copy
// move the current map location by 25 %
double dY = MapView.FinalEnvelope.Height * 0.25;

// Setting the new map location.
MapView.SetXYZ(MapView.FinalX, MapView.FinalY - dY, MapView.FinalZoom, Map.UseAnimation);

Moving the map 25% to the right:

C#
Copy
// move the current map location by 25 %
double dX = MapView.FinalEnvelope.Width * 0.25;

// Setting the new map location.
MapView.SetXYZ(MapView.FinalX + dX, MapView.FinalY, MapView.FinalZoom, Map.UseAnimation);

Moving the map 25% to the left:

C#
Copy
// move the current map location by 25 %
double dX = MapView.FinalEnvelope.Width * 0.25;

// Setting the new map location.
MapView.SetXYZ(MapView.FinalX - dX, MapView.FinalY, MapView.FinalZoom, Map.UseAnimation);

Zoom in the map:

C#
Copy
// zoom one level in with animation
map.SetZoomF(map.Zoom + 1, true);

// zoom one level in without animation
map.SetZoomF(map.Zoom + 1, false);

// zoom in to an intermediate level with animation
map.SetZoomF(map.Zoom + 0.322, true);

// zoom in to an intermediate level without animation
map.SetZoomF(map.Zoom + 0.322, false);

Zoom out the map:

C#
Copy
// zoom one level out with animation
map.SetZoomF(map.Zoom - 1, true);

// zoom one level out without animation
map.SetZoomF(map.Zoom - 1, false);

// zoom out to an intermediate level with animation
map.SetZoomF(map.Zoom - 0.322, true);

// zoom out to an intermediate level without animation
map.SetZoomF(map.Zoom - 0.322, false);
